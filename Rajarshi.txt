Install

Ubuntu 20.04 LTS
Nginx
php 7.4 (php-mysql extension)
Mariadb 10.3.39


Configuration Notes:

1. Git clone the repo.
2. Create & Edit appengine/w3root/clientPaths.cgi (w3path)
3. Create & Edit appengine/w3etc/LocalSettings.php (base_url & database)
4. Run setup.sh

First time on new server:

5. Nginx conf sample at appengine/nginx.conf
6. Add .cgi to php-fpm security.limit_extensions in pool.d/www.conf
7. PHP Extensions intl mbstring mysql required

Database setup:
